# 单一数据集涡旋标注与训练执行手册

## 1. 唯一源数据

- `data/processed/eddy_detection/19930101_20241231_clean.nc`

说明：
- 本任务只有一个源数据文件。
- train/val/test 是该文件上的时间索引切分，不是新增数据集。

## 2. 当前唯一链路（META4 + U-Net）

本仓库涡旋主链路固定为三步：

1. `scripts/02_preprocess_eddy.py`：单脚本完成对象识别 -> 像素 mask -> 背景显式 0 -> 赛题切分。
2. `scripts/03_train_eddy.py`：读取 `*_label_meta4_mask_bg0.nc` 训练 U-Net。
3. `scripts/06_eddy_assess.py`：在测试集输出指标与对象导出结果。

## 3. 一键预处理

```bash
python scripts/02_preprocess_eddy.py \
  --clean-nc data/processed/eddy_detection/19930101_20241231_clean.nc \
  --objects-nc data/processed/eddy_detection/labels/19930101_20241231_objects_meta4.nc \
  --mask-nc data/processed/eddy_detection/labels/19930101_20241231_label_meta4_mask.nc \
  --mask-bg0-nc data/processed/eddy_detection/labels/19930101_20241231_label_meta4_mask_bg0.nc \
  --split-out data/processed/splits/eddy_merged_time_competition.json \
  --overwrite
```

## 4. 训练

```bash
python scripts/03_train_eddy.py \
  --time-split-manifest data/processed/splits/eddy_merged_time_competition.json \
  --clean-nc data/processed/eddy_detection/19930101_20241231_clean.nc \
  --label-nc data/processed/eddy_detection/labels/19930101_20241231_label_meta4_mask_bg0.nc \
  --epochs 8 \
  --batch-size 4 \
  --out-dir outputs/eddy_detection/single_nc_train
```

## 5. 评估

```bash
python scripts/06_eddy_assess.py \
  --time-split-manifest data/processed/splits/eddy_merged_time_competition.json \
  --checkpoint outputs/eddy_detection/single_nc_train/checkpoints/best.pt \
  --out-dir outputs/eddy_detection/single_nc_train/eval_test
```

## 6. 标签格式说明

训练标签文件建议使用：
- `data/processed/eddy_detection/labels/19930101_20241231_label_meta4_mask_bg0.nc`

最小字段：
- `eddy_mask(time, latitude, longitude)`
- 编码：`0=background, 1=cyclonic, 2=anticyclonic`

## 7. 兼容性说明（META4 标注依赖）

`py-eddy-tracker` 对依赖版本敏感，推荐隔离环境执行：

```bash
python -m venv .venv-meta4
source .venv-meta4/bin/activate
python -m pip install -U pip
python -m pip install -r requirements.txt
```

若出现 `numpy.float_` 或轮廓对象接口相关报错，优先检查 `numpy` 与 `matplotlib` 版本是否符合项目约束。
